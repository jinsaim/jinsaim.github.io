이 페이지는 VS Code에서 작성되고 GitHub Pages를 통해 배포된 웹 페이지입니다.

✨ 주요 기능

  • 반응형 웹 디자인

  • GitHub를 통한 버전 관리
  • 무료 웹 호스팅

🚀 시작하기

VS Code와 GitHub를 연동하여 웹 페이지를 만드는 것은 매우 간단합니다:

  1. Git 저장소 초기화
  2. GitHub 저장소 생성 및 연결
  3. HTML 파일 작성
  4. GitHub에 푸시
  5. GitHub Pages 활성화
